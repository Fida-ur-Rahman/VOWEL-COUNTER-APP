# Vowel Counter App 

A simple application that counts the number of vowels in any given text.  
Built to practice string manipulation, logic implementation, and user-friendly UI design.

## Features
- Count vowels (a, e, i, o, u) in real-time
- Simple and intuitive interface
- Lightweight and fast
- Ideal for learning string handling in JavaScript/other languages

## Tech Stack
- HTML
- CSS
- JavaScript *(update if using React, Flutter, or other frameworks)*

## Purpose
Created for practice and learning, this project helps understand text processing and basic front-end development.
